``` # ¿Qué es EcoStep? EcoStep es una aplicación móvil creada para mejorar la experiencia de turistas, visitantes y habitantes dentro del mercado de Tepoztlán y el mercado nuevo. Su objetivo es facilitar la ubicación de puestos, ayudar a conocer precios aproximados, recomendar lugares y mejorar la accesibilidad para todas las personas. La aplicación busca impulsar el comercio local mediante tecnología, ofreciendo una experiencia más organizada, segura e inclusiva. # Problema que queremos resolver Muchas personas que visitan el mercado de Tepoztlán y el mercado nuevo tienen dificultades para encontrar puestos específicos, conocer precios aproximados o ubicarse fácilmente dentro del lugar. Además, algunos turistas extranjeros enfrentan problemas con el idioma, y adultos mayores o personas con discapacidad pueden tener dificultades para orientarse. # Nuestra solución EcoStep ofrece herramientas innovadoras para resolver estos problemas: 🗺️ Mapa interactivo del mercado 💰 Consulta de precios aproximados 🔎 Buscador de productos y puestos ♿ Modo inclusivo con letras grandes y audio guía 🌍 Traducción español-inglés para turistas ⭐ Recomendaciones de productos y zonas populares # ¿Cómo gana dinero EcoStep? EcoStep puede generar ingresos de diferentes maneras: * Comerciantes destacados dentro de la app * Publicidad de negocios locales * Versión premium para turistas * Convenios con comercios y espacios turísticos # Público objetivo * Turistas nacionales e internacionales * Habitantes locales * Comerciantes del mercado * Adultos mayores * Personas con discapacidad # Beneficios ## Para turistas ✔️ Mejor orientación dentro del mercado ✔️ Ahorro de tiempo ✔️ Información clara de precios ✔️ Mayor seguridad y comodidad ## Para comerciantes ✔️ Más clientes potenciales ✔️ Mayor visibilidad de sus negocios ✔️ Promoción dentro de la app # Misión Mejorar la experiencia turística y comercial mediante una aplicación tecnológica, inclusiva y accesible para todos los visitantes del mercado de Tepoztlán y el mercado nuevo. # Visión Convertir a EcoStep en una plataforma reconocida en México, expandiendo el proyecto a otros mercados turísticos y espacios comerciales. # ¿Por qué invertir en EcoStep? EcoStep combina turismo, tecnología, accesibilidad y apoyo al comercio local en una sola plataforma, con potencial de crecimiento y expansión a otras ciudades.
